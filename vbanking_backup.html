Virtual Banking App Design

About

A leading bank received virtual bank license and was setting up new digital design capabilities in order to support the launch of its mobile banking application. To handle with the increased activities in creating digital experiences, client needed to ramp up its digital design output and to seek for a new way of delivering and communicating design with product squads more effectively.

Role

  • UX Designer
  • UX Researcher

Team members

  • Jean Shen - UX Designer

  • Lok Wong - illustrator

Implement design sprint plan in Agile enviornment

In orfer to drive the holistic design environment and work with all squads effectively, We took agile approach to implement our design sprint.

1. Discover

  • Understand the scope of sprint, and development timeline for each sprint
  • Go through user stories (WHO/WHEN/WHAT/HOW) to undstand business scenarios and do sprint plan
  • Go through benchmarking research to get insights from best practice

2. Define

  • Run co-creation workshop with the Squads to do ideation.
  • Sketch out the ideas
  • Create user flows

3. Design

  • Design system
  • User interface design
  • Define illustration, animation and transition

4. Test+Iterate

  • Create clickable prototype
  • Prepare user testing plan (Goals, user tasks, observation parts )
  • Conduct user testing
  • Refine design based on testing results

1. Discovery: Gather user stories & do sprint plan

Gather user stories from Squads to understand the sprint scope.

After gathering the user stories from all squads, we do planing based on the working load of each squads in the beginning of each sprint by estimating the task weight.

2. Define: Co-creation workshop to run ideation

After going through the user stories and research insights, we run co-creation workshop to do ideation with Product Onwers and Business Analysts.

Example: Pre-Login

  • Pre-login OR login only?
  • What types of information user would like to see?

Functional

Quick action button

3. Design: Design system & UI design

Based on the 1.0 design system, We helped defining the new UI design system in order to set up a more dynamic foundation for UX design

UI Components Sample

Saving acccount details

User can set up saving goal while creating saving accounts. App simulator will help user to set the initial deposits and recurring transaction.

In saving account details page, user can track saving satus via the progress bar. We will also provide gamification features to motivate user saving more.

4. Test + iterate

After completing the user interface design, we prepare the testing plan by listing test goals, user tasks/questions and observation parts.

Example: Testing goals of Pre-Login

  • See if user prefer to have just Login or Pre-login with other functions
  • What functions user would like to have? What's the priority?
